Why Interleaved?

Blocked practice (Python for a month, then SQL for a month) feels efficient but produces fragile memory. Interleaved practice — touching all scopes every week — forces your brain to retrieve and reconnect, which is exactly what builds durable skill.

Suggested Daily Rhythm

MONSQL drill — 15–30 min + weekly deep task
TUEPython — scripting, pipeline, or library
WEDDeep focus — dbt / Airflow / Snowflake
THUDevOps — Docker / Terraform / CI-CD
FRIProject work — integrate the week
WEEKENDLeetCode SQL, blog, or rest — no grinding

On Hours

Each week totals ~20–28 hours. If you have only 10 hrs/week, halve the tasks and run the roadmap over 48 weeks. The structure matters more than the speed. Consistency beats intensity.

Weekly Handoff Note
